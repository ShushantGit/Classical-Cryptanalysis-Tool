# Classical Cryptanalysis Tool

## Overview

A C-based cryptanalysis toolkit designed to demonstrate classical substitution cipher techniques, frequency analysis, encryption, and decryption.

## Features

- Substitution Cipher Encryption
- Frequency Analysis of Ciphertext
- Custom Character Mapping
- Ciphertext Decryption
- File-Based Processing

## Project Workflow

```text
Plain Text
    ↓
Encryption
    ↓
Ciphertext
    ↓
Frequency Analysis
    ↓
Character Mapping
    ↓
Decryption
    ↓
Recovered Plain Text
```

## Technologies Used

- C Programming
- File Handling
- Character Manipulation
- Frequency Analysis
- Classical Cryptography
